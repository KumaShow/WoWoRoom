> 後台: [點我](https://kumashow.github.io/WoWoRoom/admin)

## 網站功能

1. Axios 串接遠端伺服器 API，取得產品資料
2. C3.js 圖表使用，用於後台顯示訂單相關資料
3. 送出訂單時的表單驗證
4. 商品可以加入購物車

## 使用技術

1. JavaScript ES6
2. C3.js
3. Axios
4. Gulp + EJS
5. Bootstrap 5 & SCSS
